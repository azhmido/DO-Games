# 🎮 DO-Games

**DO-Games** adalah platform eksplorasi game premium berbasis **React**, dirancang untuk membantu pengguna menemukan berbagai game *free-to-play*, menyimpan koleksi favorit, hingga berinteraksi melalui ulasan komunitas.  
Antarmuka dibuat modern dengan animasi halus menggunakan **Framer Motion**, serta mendukung **multi bahasa (IND/EN)** untuk pengalaman global.

---

## 🚀 Fitur Utama

### 🔐 Autentikasi Pengguna
- Registrasi & login aman  
- Protected Routes  
- Manajemen pengguna dengan Context API  

### 🏠 Beranda (`/`)
- Hero Carousel interaktif  
- Rekomendasi game populer  
- Daftar game yang sedang trending  

### 🔑 Login (`/login`) & Register (`/register`)
- Validasi form  
- Navigasi responsif  
- UI minimalis dan modern  

### 👤 Profil (`/profile`)
- Edit bio & lokasi  
- Lihat informasi akun  
- Sistem **Gaming Level Progress**  

### 🔎 Jelajahi (`/explore`)
- Smart Discovery (pencarian cerdas)  
- Filter genre, platform, dan tingkat kesulitan  

### 🎮 Daftar Game (`/games`)
- Katalog lengkap  
- Pencarian cepat  
- Filter genre  
- Mode tampilan **Grid / List**  

### 📘 Detail Game (`/games/:id`)
- Informasi lengkap game  
- Komentar/ulasan komunitas  
- Spesifikasi sistem & deskripsi panjang  
- Ruang diskusi komunitas  

### ⭐ Favorit (`/favorites`)
- Simpan game favorit  
- Kelola koleksi pribadi  

### 🌐 Multi Bahasa (ID/EN)
- Sistem translasi dinamis  
- Ganti bahasa instan  
- Integrasi penuh dengan Context API

## 🛠️ Teknologi yang Digunakan

| Teknologi       | Fungsi                               |
|-----------------|----------------------------------------|
| **React.js**    | UI berbasis komponen                  |
| **React Router**| Navigasi & proteksi rute              |
| **Framer Motion** | Animasi transisi interaktif         |
| **Context API** | Manajemen state global                |
| **Tailwind CSS** | Styling modern & responsif           |
| **Lucide React** | Ikon vektor profesional              |
